# Temperature Conversion

A simple web app that converts temperatures between Celsius and Fahrenheit.

## Features

- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Clean, responsive UI with instant results

## Project Structure

```
├── index.html   # App markup and layout
├── index.css    # Styling
└── index.js     # Conversion logic
```

## How to Use

1. Clone or download the repository
2. Open `index.html` in your browser — no build steps or dependencies required
3. Enter a temperature value in the input field
4. Select a conversion direction using the radio buttons
5. Click **Submit** to see the result

## Conversion Formulas

- **Celsius → Fahrenheit:** `(°C × 9/5) + 32`
- **Fahrenheit → Celsius:** `(°F − 32) × 5/9`

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
